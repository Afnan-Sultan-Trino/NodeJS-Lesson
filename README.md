## 📌 Overview

This repository contains my hands-on practice code for understanding **Local Modules** in Node.js. The goal of this phase was to learn how to create custom JavaScript files (modules), export functions/variables from them, and import them into other files using Node.js's built-in `require()` function.

This project simulates a simple student profile system to demonstrate the core concepts of modular programming in a Node.js environment.
